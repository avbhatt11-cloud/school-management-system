# 🏫 School Management System

A complete web application built with ASP.NET MVC for managing school operations.

## ✨ Features
- 👨‍🎓 Student registration & management
- 👨‍🏫 Teacher management
- 📋 Attendance tracking
- 📊 Result & marks management
- 💰 Fee management with online payment (Cashfree)
- 🔐 Login & authentication system

## 🛠️ Technologies Used
- **Backend:** ASP.NET MVC, C#
- **Database:** SQL Server, Entity Framework
- **Frontend:** Bootstrap 5, jQuery, HTML, CSS
- **Payment:** Cashfree Payment Gateway
- **Tools:** Visual Studio 2022, Git
